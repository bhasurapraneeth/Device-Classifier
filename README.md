# Device Classifier Android App

This Android application uses a TensorFlow Lite model to classify device images into five categories. Users can select an image from their gallery, and the app predicts the device type with confidence.

## Features

- Select an image from the device gallery
- Run inference using a TFLite model
- Display predicted device class and confidence
- Simple and user-friendly interface

## Technologies

- Java & Kotlin
- Android SDK
- TensorFlow Lite
- Gradle
